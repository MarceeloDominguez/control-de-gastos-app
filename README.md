## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/control-de-gastos-app.git
```

Go to the project directory

```bash
  cd control-de-gastos-app
```

Install dependencies

```bash
  npm install
```

Create the Project

```bash
  npx expo run:android 
```

Start the server

```bash
  npx expo start --dev-client
```
